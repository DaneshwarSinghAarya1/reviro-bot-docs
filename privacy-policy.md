Privacy Policy for Reviro Bot

1. Introduction
Welcome to Reviro Bot! Your privacy is important to us. This Privacy Policy explains how we collect, use, and protect your data when you interact with our bot on Discord.

2. Data We Collect
Reviro Bot may collect the following data:

User Data: Your Discord username, user ID, and messages sent to the bot.

Server Data: Server name, ID, and roles for moderation purposes.

3. How We Use Your Data
We use the collected data to:

Provide bot functionalities such as moderation and review management.

Ensure a smooth and secure user experience.

Improve the bot’s performance and features.

4. Data Sharing & Security

We do not sell, trade, or share your data with third parties.

All collected data is stored securely and used only for Reviro Bot’s functionalities.

Only authorized personnel have access to the stored data.

5. Data Retention

User and server data is retained as long as necessary to provide bot services.

Review data is stored until the server owner decides to remove it.

Users can request data deletion by contacting the server administrator.

6. User Rights
As a user, you have the right to:

Request access to your data.

Request the deletion of your data.

Stop using the bot at any time to prevent further data collection.

7. Changes to This Policy
We may update this Privacy Policy from time to time. Updates will be announced on the bot’s official support server.

8. Questions & Support
If you have any questions about this Privacy Policy, please contact the server administrator or visit the bot’s support server.

By using Reviro Bot, you agree to this Privacy Policy.
